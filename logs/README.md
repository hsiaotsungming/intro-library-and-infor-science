# Project Logs

This folder records the long-term development of the Intro to Information and Library Science project.

## Structure

- `entries/`: dated work logs for what changed during each work session.
- `decisions/`: durable project decisions, especially decisions that affect structure, publishing, design, workflow, or bilingual handling.

## Rule

Use concise entries. Record enough context to recover the work later, but keep publishable website files in `site/` and reusable project rules in `project-config/`.
